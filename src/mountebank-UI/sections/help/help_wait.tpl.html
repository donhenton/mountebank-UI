Help for Wait

If a number is passed in, mountebank will wait that number of milliseconds before returning. If a string is passed in, it is expected to be a parameterless JavaScript function that returns the number of milliseconds to wait.

Wait function sample

function() { 
    return Math.floor(Math.random() * 901) + 100; 
}

Note that the Mountebank server must run with the --allowInjection flag. Mountebank decoration documentation
